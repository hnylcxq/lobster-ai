# 🤝 Contributing to LobsterAI

Thank you for considering contributing to LobsterAI!

## 📋 Code of Conduct

Please be respectful and constructive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).

## 🚀 How to Contribute

### 1. Fork the Repository

Click the "Fork" button on GitHub.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/lobster-ai.git
cd lobster-ai
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Changes

Make your changes and ensure tests pass.

### 5. Commit Your Changes

```bash
git add .
git commit -m 'Add some amazing feature'
```

### 6. Push to GitHub

```bash
git push origin feature/your-feature-name
```

### 7. Open a Pull Request

Go to the original repository and click "New Pull Request".

## 📝 Pull Request Guidelines

- Follow the existing code style
- Write clear commit messages
- Include tests if applicable
- Update documentation if needed

## 🐛 Reporting Bugs

Use GitHub Issues to report bugs. Include:
- Clear description
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## 💡 Feature Requests

Open an issue with:
- Clear feature description
- Use cases
- Mockups or examples (if applicable)

---

**Thank you for contributing! 🎉**
